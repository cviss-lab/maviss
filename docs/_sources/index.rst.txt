MAVISS
======

.. raw:: html

   <p align="center"> <img src="_static/mav_field.jpg" alt="MAVISS Drone" style="max-width: 340px; height: auto;"> </p>

**MAVISS** (Modular Aerial Vehicle with Intelligent Sensor Systems) is a modular and intelligent drone platform focused on:

- Real-time perception and mapping  
- Onboard autonomous navigation  
- Integration with ROS 2, PX4, and Nvidia Jetson  

----

.. raw:: html

   <p>
     This project is developed under the 
     <a href="https://cviss.net/" target="_blank" rel="noopener noreferrer">
       CVISS@UWaterloo
     </a>, focusing on computer vision, intelligent sensing, and autonomous systems.
   </p>
